# Movie Theater APP

[*Movie Theater Link*](http://movie-booking.herokuapp.com/)

## To run local:

  1. Connect to the database (with pgAdmin)

  2. touch .env file and set up GMAIL_USERNAME and GMAIL_PASSWORD variables

  3. Run: $ rails s
